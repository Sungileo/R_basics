반복문으로 3 배수 출력

x <- c() #비어있는 벡터x 생성
for (i in 1:5) {
  j <- i*3
  x <- c(x,j)#벡터 X에 j의 값 추가
  }
print(x)
[1]  3  6  9 12 15

반복문으로 i, x 출력 (3배수)

  • cat 활용 (print() 사용시 \n은 문자열로 출력됨)

  • \n : 줄바꿈 기호

x <- c() 
for (i in 1:9) {
  j <- i*3
  x <- c(x, i, j, "\n")
}

cat(x)

반복문, 조건문 같이쓰기

1,2,3,4,5 중 2로 나누었을때 나머지가 1인것 (홀수) 만 출력

x <- c(1,2,3,4,5)

for(i in x){
  if(i%%2==1) {
    print(i)
  }
}
[1] 1
[1] 3
[1] 5

1,2,3,4,5 중 2로 나누었을때 나머지가 1인것 (홀수) 만 x에 넣기

x <- c()
for (i in 1:5){
  if(i%%2==1){
    x <- c(x,i)
  }
}
print(x)
[1] 1 3 5

반복문 안썼을때

x <- c()

i <- 1
if(i%%2==1) {

}

구구단

#반복문 notion 문제 1 과제

result <- data.frame()

for (i in 2:9){
  for(j in 1:9) {
    k <- i*j
    result_temp <- c(i,j,k)
    result <- rbind(result,result_temp)
  }
}

DT::datatable(result)
