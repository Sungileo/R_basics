8. ( : Travel_Plan.csv) 파일명 제리 스티븐슨 은 (Jerry Stevenson) 여행사의 매니저이다.

그는 고객이 내년 안에 여행할 것인지 여부를 예측할 수 있는 모형을 구축하고자 한다 그는 . 다음과 같은 변수를 포함하는 데이터셋을 수집했다 :

  • 대학학위를 가지고 있는지 (College)

  • 신용카드를 가지고 있는지 (Credit-Card)

  • 음식에 대한 연간 가계지출 달러 (FoodSpend)

  • 연소득 달러 고객이 (Income)

  • 내년 안에 여행할 계획이 있는지 여행계획 (TravelPlan : 1 = 있음, 0 = 없음)

데이터의 일부를 아래 표에 보여준다

a) Travel_Plan.csv R 파일 를 로 불러와서 변수 df 로 저장하세요

df <- read.csv("Travel_Plan.csv")

b) df 변수 의 속성을 확인하세요. 불러온 데이터는 어떤 형테의 자료인가요 매트릭스인가요, 데이터프레임 인가요

(반드시 속성을 확인하는 함수 사용하세요)

is.matrix(df)
is.data.frame(df)

c) 50,000 연소득이 달러 이상인 사람은 몇 명인가요 점

nrow(subset(df,Income > 50000))

d) (Credit-Card) 신용카드 가 있는 사람 중에 연소득이 60,000 달러 이상인 사람은 몇명인가요

nrow(subset(df,CreditCard == "Yes" & Income > 60000))

e) 여행계획이 있는 사람들의(TravelPlan : 1) 연소득 평균값과 여행 계획이 없는 사람들의(TravelPlan : 0) 연소득 평균값을 계산한 다음 비교하세요

여행계획이 있는 사람의 연소득 평균값이 얼마 더 높은가요

mean(subset(df,TravelPlan == 1)$Income)
mean(subset(df,TravelPlan == 0)$Income)

f) (Income)연소득을 기준으로 내림차순으로 정렬한 다음 연소득 상위 100명의 데이터를추출하세요

(변수명 : df_100 ) 추출된 데이터를 Travel_Plan_100.csv 이름으로 저장하세요
